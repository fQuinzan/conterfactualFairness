# conterfactualFairness

Repository for the Counterfactual Fairness Experiments

## Files and forlders

- main.py : file to run the experiment to compare performance with HSCIC and without HSCIC. 
- model.py : it contains the class to define the deep learning model; the architecture and the parameters are not tuned to best performance.
- HSCIC.py : it contains a class for the HSCIC loss.
